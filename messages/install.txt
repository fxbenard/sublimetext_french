French Menu for Sublime text 3

Sublime Text 3 is a French plug-in.
It is based on the Japanese file.

Application procedure
1． C:\Users\ User name Iappdata text 3 \ Packages installed * FR files,
C:\Users\ User name Iappdata text 3 \ Packages
Copy it to. * If there is no default folder, please create it.
2． Replaces the copied file with the original file (FR file). (Just in case, if you have the original file, please keep it with org. ）
3． C:\Users\ User name Iappdata text 3 \ PackagesMenu (without FR)
C:\Users\ User name Iappdata text 3 \ Packages
Copy it to. The top menu that has been overwritten by the other plug-in is also Japanese.
